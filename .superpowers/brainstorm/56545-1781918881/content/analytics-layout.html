Phase 3.0 — Design Choice 1 of 5

Analytics Page Layout — How should the new Analytics page be structured?

A
┌─ ANALYTICS ──────────────────────────────────┐ │ [Portfolio] [Agents] [Corr] [Attrib] [Backtest]│ ├───────────────────────────────────────────────┤ │ │ │ (content of selected tab renders here) │ │ │ │ Each tab = full-page focused view │ │ │ └───────────────────────────────────────────────┘

A — Tabbed

Each section (Portfolio, Agents, Correlation, Attribution, Backtest) gets its own tab. Clean, one thing at a time. Best if you want focused views without scroll.

Pros

  • Uncluttered, no scroll
  • Fast tab-switch to any section

Cons

  • Can't see everything at once
  • More clicks to compare sections
B
┌─ ANALYTICS ─────────────────────────────────┐ │ PORTFOLIO PERF [30d ▼] [YTD] [ALL] │ │ ┌─────────────────────────────────────────┐ │ │ │ Equity Curve (Recharts LineChart) │ │ │ └─────────────────────────────────────────┘ │ │ [+12.3%] [Sharpe 1.4] [DD -8.3%] [Win 67%] │ │ │ │ AGENT LEADERBOARD [Sort: Return ▼] │ │ Stock Picker +14% ██████ Sharpe 1.8 │ │ Energy Trader +10% █████ Sharpe 1.3 │ │ │ │ CORRELATION MATRIX (heatmap) │ │ ATTRIBUTION — Today (waterfall) │ │ BACKTEST (runner form + results) │ └──────────────────────────────────────────────┘

B — Single Scroll

All sections on one page, stacked. Scroll to see Equity → Leaderboard → Correlation → Attribution → Backtest. Like a dashboard report.

Pros

  • Everything visible in one scroll
  • Dashboard-report feel

Cons

  • Long page, lots of scroll
  • Backtest buried at bottom
C
┌─ ANALYTICS ──────────────────────────────────┐ │ │ │ LEFT PANEL (agents) │ RIGHT PANEL (detail)│ │ ───────────────── │ ─────────────────── │ │ ● All Agents ✅ │ Equity curve │ │ › Stock Picker $14% │ Sharpe / DD / Win │ │ Energy $10% │ Attribution detail │ │ News $6% │ │ │ Index $4% │ [Run Backtest ▶] │ │ │ │ └──────────────────────────────────────────────┘

C — Split Pane

Agent list on left, detail panel on right. Click any agent to drill into its equity curve, correlation, attribution. Similar to email client layout.

Pros

  • Great for per-agent drill-down
  • No scroll needed

Cons

  • Portfolio-level view awkward
  • Correlation matrix hard to fit

Click to select. You can change your mind before hitting Next. More choices follow for each chart style.
