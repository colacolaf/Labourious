A — Inline Panel

Backtest section lives at the bottom of the Analytics scroll page. Run it without leaving the page. Results render inline below the form.

Pros

  • No context switch — stays in Analytics
  • Can scroll up to compare live vs backtest

Cons

  • Page gets long when results shown
B
In AgentInspector → Performance tab

full-screen overlay modal, dimmed background

B — Modal from Inspector

Button in AgentInspector's Performance tab opens a full-screen modal overlay. Backtest is agent-specific, results in the modal, close to return.

Pros

  • Agent-focused — you're already looking at that agent
  • Full screen = more chart space

Cons

  • Can't run cross-agent comparison from Analytics page
  • Must open inspector first
