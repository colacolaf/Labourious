A — Waterfall Chart

Stacked waterfall: start → each agent adds/subtracts → end total. Classic financial attribution view. Shows flow clearly. Built with Recharts BarChart.

Pros

  • Shows cumulative flow visually
  • Standard financial chart type

Cons

  • Harder to compare magnitudes side-by-side

P&L ATTRIBUTION — 2026-06-19  Total: +$684

B — Horizontal Bar (% contribution)

Each agent = one bar showing its % of total daily P&L. Green bars = positive, red = drag. Total shown top-right. Easy ranking at a glance.

Pros

  • Instant rank comparison
  • % and $ both visible
  • Cleaner on small screens

Cons

  • Doesn't show cumulative flow
