Correlation Matrix — How should agent correlations be displayed? (Colors match the real retro-professional palette: teal/amber/cyan on dark)

dark teal = low corr · bright teal = high corr · amber ⚠ if > 0.7

A — Color Heatmap

Cells colored by correlation strength. Teal intensity = correlation level. Diagonal always brightest. High correlation (>0.7) flags amber ⚠.

Pros

  • Instantly see clusters visually
  • Professional Bloomberg-style

Cons

  • Color alone can be ambiguous
B
CORRELATION — LAST 30D DAILY RETURNS
News Energy Stock Index
News — 0.18 0.09 0.31
Energy 0.18 — 0.42 0.11
Stock 0.09 0.42 — 0.28
Index 0.31 0.11 0.28 —
⚠ High correlation (>0.7) flagged amber — none here

B — Numbers Only

Plain table with exact values. Zebra rows for readability. High correlation (>0.7) gets amber ⚠ flag. Maximally precise, no color ambiguity.

Pros

  • Exact values, no guessing
  • Simpler to implement

Cons

  • Harder to spot patterns at a glance
