# Penthouse — The Top

> Calm. Quiet. The Portfolio Manager sits at the center. The PM Bodyguard stands by the window, watching. This is where everything comes together.

## Occupants

| Agent | Role |
|-------|------|
| **Portfolio Manager** | The main orchestrator. The ONLY agent the user directly chats with. Receives room-level syntheses from all 13 leads, produces the final report + summary + action options. Calm, collected, deliberate. |
| **PM Bodyguard** | Stands beside the Portfolio Manager at all times. Last line of defense — if the PM is about to make a catastrophic error, the bodyguard speaks up. Loyal, protective, speaks only when it matters. |

## Interaction

- User enters the Penthouse via elevator from Floor 4.
- User sits across from the PM. This is the chat interface.
- The PM Bodyguard is visible but silent. Only interrupts for critical warnings.
- From here, the user can see a dashboard of all active rooms and agents below.
